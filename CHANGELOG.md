# Changelog

## 2026-04-23
- Кнопки действий в формах и инспекторах приведены к одному правилу: `Сбросить`, `Сохранить`, `Применить` и удаление перенесены в нижние action-зоны блоков.
- Скролл у левой панели возвращен внутрь карточки `Входные данные`, а расчет `max-height` исправлен на более точный вариант от `100dvh`, чтобы не появлялось ложное переполнение.
- Лишний скролл у панели ввода убран: ограничение высоты и прокрутка переносились с карточки формы на контейнер левой колонки, чтобы скролл появлялся только при реальном переполнении окна.
- У панели входных данных убран правый градиентный разделитель, а внутренний скролл сделан тоньше и спокойнее визуально.
- Для длинной левой панели ввода добавлен внутренний скролл по высоте окна на desktop, чтобы до нижних полей и кнопки сброса всегда можно было добраться.
- Для корректной работы `sticky` горизонтальное скрытие переписано с `overflow-x: hidden` на `overflow-x: clip` у `html/body`, чтобы не создавать лишний scroll-контейнер.
- Исправлена основная причина некорректного `sticky` у левой панели: убран `overflow-x: hidden` с `.app`, который мешал блоку закрепляться при прокрутке.
- Левая панель ввода временно переводилась в отдельную фиксированную desktop-колонку для проверки поведения при длинной прокрутке.
- Панель ввода переработана: обновлены типографика, компоновка и нижняя кнопка сброса.
- Layout калькулятора перестроен под живую работу: на десктопе форма стала левой рабочей колонкой, а на мобильном идет первой перед итогами и графиком.
- На странице калькулятора убрано переключение между режимами: результаты, график и план по годам теперь видны сразу, а левая колонка стала навигацией по секциям.
- Удален footer, а левая навигация переведена в фиксированную колонку на всю высоту окна на десктопе с адаптивным откатом на мобильных.
- Tailwind удален из проекта: `ui`-слой переписан обратно на Sass/CSS с тем же API компонентов, а Vite переведен на modern Sass API без предупреждения `legacy-js-api`.
- Проект переведен с локальных shadcn-like primitives на реальные `shadcn/ui`: подключены Tailwind/shadcn зависимости, официальный `ui`-слой и совместимость с текущими страницами.
- Добавлен общий footer с информацией о проекте, дисклеймером по расчетам, localStorage и ссылками на разделы.
- Подключен shadcn-style UI layer с локальными primitives и переписаны основные панели, поля и навигация на новые UI-компоненты.

## 2026-04-22

- В `AGENTS.md` добавлено правило использовать ExecPlan для сложных функций и значимых рефакторингов.
- Добавлен `AGENTS.md` с описанием стека, структуры проекта, команд, правил кода и самопроверки.
- Добавлена пенсионная страница с расчетом нужного капитала, срока выплат и мягкими рекомендациями по возрасту.
- Добавлена навигация между страницами через hash-ссылки, чтобы разделы можно было открывать напрямую.
